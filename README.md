Egy egyszeru To-do app igazabol semmi kulonos 


Funkciók
- Feladatok hozzáadása (szövegmezőből megy a ToDoTask obj. listába)
- Checkbox rendszer
- Törlés (a lista utolsó eleme)

Részletek
- Swift 5
- SwiftUI
- Custom Task modell (`Identifiable`, `UUID`)
- Reusable `ButtonStyle`
